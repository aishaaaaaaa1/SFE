Dossier des PDF de réglementation
=================================

Les fichiers .pdf listés ici sont servis en téléchargement depuis les pages
reg-dahirs.html, reg-decrets.html, reg-circulaires.html et reg-arretes.html.

Pour mettre à jour un texte officiel : remplacez le fichier portant le même nom,
ou modifiez le nom du fichier dans scripts/generate-pages.mjs puis exécutez :
  npm run pages

Les sources brutes peuvent être regroupées dans le dossier pdf/ à la racine du
projet, puis copiées ici avec des noms stables (voir scripts/generate-pages.mjs).

La commande npm run reg-pdfs ne génère plus de modèles par défaut (évite
d’écraser les PDF réels).
